# Triathlon Training Platform

A modern web application for creating and managing triathlon training programs. Built with Next.js, TypeScript, and Tailwind CSS.

## Features (Current & Planned)

- 📅 Interactive calendar interface for workout planning
- 🏊‍♂️ Support for multiple training types (swim, bike, run)
- 📊 Training load visualization
- 🔄 Strava integration (planned)
- 📱 Responsive design for all devices

## Tech Stack

- [Next.js](https://nextjs.org/) - React framework for production
- [TypeScript](https://www.typescriptlang.org/) - Static type checking
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [date-fns](https://date-fns.org/) - Modern JavaScript date utility library
